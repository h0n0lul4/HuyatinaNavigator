# Хуятина Навигатор

Небольшой браузер, вдохновлённый классическими браузерами конца 90-х и начала 2000-х годов, но способный открывать современные веб-страницы. Написан на Python с использованием PySide6 и Qt WebEngine.

Проект создаётся в первую очередь как способ научиться программированию. Вместо того чтобы делать очередной калькулятор, я решил написать собственный браузер и постепенно развивать его, добавляя новые возможности с каждым релизом.

## Текущие возможности

* Открытие веб-страниц
* Адресная строка
* Переход по Enter
* Кнопки «Назад» и «Вперёд»
* Кнопка «Домой»
* Кнопка обновления страницы
* Настраиваемая домашняя страница (`settings.json`)
* Окно «О программе»
* Вкладка «Special Thanks»

## В планах

* Несколько окон вместо вкладок (в стиле старых браузеров)
* Полноценное окно настроек
* Анимированный логотип во время загрузки страниц
* Улучшение интерфейса
* Собственные иконки и графика
* Готовые `.exe`-релизы в ZIP-архивах
* Английский
* Постепенное развитие архитектуры проекта
