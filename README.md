Хуятина Навигатор

Небольшой браузер косящий под старые браузеры того времени но способный октрывать современные страницы, написанный на Python с использованием PySide6.

Проект создаётся в первую очередь как способ научиться программированию. 
Вместо того чтобы делать очередной учебный калькулятор, я решил написать собственный браузер и постепенно развивать его.

Текущие возможности:

- 🌐 Открытие веб-страниц
- 🔎 Адресная строка
- ⬅️➡️ Кнопки «Назад» и «Вперёд»

В планах:
- Закладки
- История
- Загрузки
- Настройки
- Улучшение интерфейса
- Собственный стиль браузера
